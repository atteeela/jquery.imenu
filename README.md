## Installation

Include jQuery and `jquery.imenu.min.js` onto your page:

```html
    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <script src="jquery.imenu.min.js"></script>
```
